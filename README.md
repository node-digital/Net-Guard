
# NetGuard

A Easy Way To Manage Your Network
## Features

- On The Fly Server Networking
- ipconfig editing
- rate limiting 
- ip whitelisting
- ASN blocking
- Not Needed Port Blocking
- ICMP Blocking


## Setup

```bash
  sudo python3 main.py
```
